How to use the tool 
- Place the 'ScreenGrab_V1.0Alpha.jar' in the folder necessarry to capture & store the screenshots.
- Launch the 'ScreenGrab_V1.0Alpha.jar' for manual mode.
- Use Run.bat to capture from other sw(Automated mode).
